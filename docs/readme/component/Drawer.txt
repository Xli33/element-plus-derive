<h2>Drawer</h2>
<p>默认带footer的el-drawer，更便于控制易于使用</p>
<ul>
<li><code>0.0.5+</code><br>统一默认的关闭行为，通过图标<code>X</code>以及点击遮罩时尝试触发 <code>cancel</code> 事件，若未监听该事件则直接关闭，效果同点击footer的取消按钮</li>
</ul>

<h2>props</h2>
<p><code>modelValue</code> <em>Boolean</em><br>双向绑定</p>
<p><code>has-footer</code> <em>Boolean</em> （default <code>true</code>）<br>是否显示footer</p>
<p><code>footer-attrs</code> <em>Object</em><br>传递给ModalFooter的props</p>
<h2>emits</h2>
<p><code>ok</code><br>点击确定按钮触发</p>
<p><code>cancel</code><br>点击取消按钮触发。未监听该事件时会直接关闭Drawer</p>
<ul>
<li><code>0.0.5+</code>
点击遮罩与关闭icon时也会触发该事件，若需分开处理可以使用 el-drawer 的 <code>before-close</code></li>
</ul>
<h2>slots</h2>
<p><code>header()</code><br>header</p>
<p><code>default()</code><br>body</p>
<p><code>footer()</code><br>footer</p>
